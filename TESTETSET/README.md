# 📝 シンプル掲示板アプリケーション

Flask + HTML/CSS/JavaScriptで作成したシンプルな掲示板アプリケーションです。

## 🌟 機能

- ✅ 投稿の一覧表示
- ✅ 新規投稿の作成
- ✅ 投稿の編集
- ✅ 投稿の削除
- ✅ タイムスタンプの自動記録
- ✅ レスポンシブデザイン
- ✅ フラッシュメッセージ
- ✅ フォームバリデーション
- ✅ 下書き自動保存（ローカルストレージ）

## 📋 必要要件

- Python 3.8以上
- pip（Pythonパッケージマネージャー）

## 🚀 セットアップ方法

### 1. 依存パッケージのインストール

```bash
pip install -r requirements.txt
```

または個別にインストール：

```bash
pip install Flask==3.0.0 Werkzeug==3.0.1
```

### 2. アプリケーションの起動

macOSの場合（`python`コマンドが使えない場合）：

```bash
python3 app.py
```

その他のシステム：

```bash
python app.py
```

### 3. ブラウザでアクセス

アプリケーションが起動したら、ブラウザで以下のURLにアクセスしてください：

```
http://localhost:8080
```

または

```
http://127.0.0.1:8080
```

**注意:** macOSでポート5000が使用中の場合、ポート8080を使用します。

## 💡 使い方

### 新規投稿の作成

1. トップページの「新規投稿」ボタンをクリック
2. タイトル、投稿者名、内容を入力
3. 「投稿する」ボタンをクリック

### 投稿の編集

1. 投稿カードの「編集」ボタンをクリック
2. 内容を修正
3. 「更新する」ボタンをクリック

### 投稿の削除

1. 投稿カードの「削除」ボタンをクリック
2. 確認ダイアログで「OK」をクリック

## 🎨 特徴

- **モダンなUI**: グラデーション背景とカードデザイン
- **レスポンシブ**: モバイルデバイスにも対応
- **アニメーション**: スムーズなトランジション効果
- **ユーザーフレンドリー**: 直感的な操作性
- **データ永続化**: JSONファイルでデータを保存

## 🔧 カスタマイズ

### 秘密鍵の変更

`app.py`の以下の行を変更してください：

```python
app.secret_key = 'your-secret-key-here'  # ランダムな文字列に変更
```

### ポート番号の変更

`app.py`の最後の行を変更してください：

```python
app.run(debug=True, host='0.0.0.0', port=5000)  # portを変更
```

### スタイルのカスタマイズ

`static/css/style.css`を編集してデザインを変更できます。

## 📝 データ構造

投稿データは`posts.json`に以下の形式で保存されます：

```json
[
  {
    "id": 1,
    "title": "投稿タイトル",
    "content": "投稿内容",
    "author": "投稿者名",
    "created_at": "2025-12-23T10:30:00",
    "updated_at": "2025-12-23T10:30:00"
  }
]
```

## 🐛 トラブルシューティング

### Flaskがインストールされていない

```bash
pip install Flask
```

### ポートが既に使用されている

別のポート番号を指定してください：

```bash
# app.pyを編集してポート番号を変更
```

### posts.jsonが見つからない

アプリケーション起動時に自動的に作成されます。手動で作成する場合：

```bash
echo "[]" > posts.json
```
